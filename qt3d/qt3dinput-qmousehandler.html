详细描述

Provides a means of being notified about mouse events when attached to a QMouseDevice instance.

注:QMouseHandler components shouldn't be shared, not respecting that condition will most likely result in undefined behaviors.

参照 QMouseDevice.

属性文件

containsMouse : const bool

Holds true if the QMouseHandler currently contains the mouse.

注:In this context, contains mean that the ray originating from the mouse is intersecting with the Qt3DCore::QEntity that aggregates the current QMouseHandler instance component.

访问函数:

bool containsMouse() const

Notifier signal:
