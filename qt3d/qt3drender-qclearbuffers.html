Member Type Documentation

enum QClearBuffers::BufferType
flags QClearBuffers::BufferTypeFlags

This enum type describes types of buffer to be cleared.

ConstantValue
Qt3DRender::QClearBuffers::None0
Qt3DRender::QClearBuffers::ColorBuffer( 1<<0 )
Qt3DRender::QClearBuffers::DepthBuffer( 1<<1 )
Qt3DRender::QClearBuffers::StencilBuffer( 1<<2 )
Qt3DRender::QClearBuffers::DepthStencilBufferDepthBuffer | StencilBuffer
Qt3DRender::QClearBuffers::ColorDepthBufferColorBuffer | DepthBuffer
Qt3DRender::QClearBuffers::ColorDepthStencilBufferColorBuffer | DepthStencilBuffer
Qt3DRender::QClearBuffers::AllBuffers0xFFFFFFFF

The BufferTypeFlags type is a typedef for QFlags<BufferType>. It stores an OR combination of BufferType values.
