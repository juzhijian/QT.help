详细描述

Provides a way of specifying values of a Uniform Block or a shader structure.

注:When subclassing and adding properties to QShaderData, note that if you need to nest an inner Qt3DRender::QShaderData, the data type of the property should be Qt3DRender::QShaderData* instead of the name of your subclass.
