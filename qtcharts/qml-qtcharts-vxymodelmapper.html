VXYModelMapper QML Type

Vertical model mapper for QXYSeries. 更多...

详细描述

VXYModelMapper allows you to use your own QAbstractItemModel derived model with data in columns as a data source for XYSeries based series. It is possible to use both QAbstractItemModel and XYSeries data API to manipulate data. VXYModelMapper keeps the series and the model in sync.

属性文件

columnCount : int

Defines the number of rows of the model that are mapped as the data for series. The default value is -1 (count limited by the number of rows in the model).


firstRow : int

Defines which row of the model contains the data for the first point of the series. The default value is 0.


model : SomeModel

The QAbstractItemModel based model that is used by the mapper. You need to implement the model and expose it to QML. Note: the model has to support adding/removing rows/columns and modifying the data of the cells.


series : XYSeries

Defines the XYSeries object that is used by the mapper. All the data in the series is discarded when it is set to the mapper. When new series is specified the old series is disconnected (it preserves its data).


xColumn : int

Defines which column of the model is kept in sync with the x values of the series. Default value is -1 (invalid mapping).


yColumn : int

Defines which column of the model is kept in sync with the y values of the series. Default value is -1 (invalid mapping).
