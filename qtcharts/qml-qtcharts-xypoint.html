XYPoint QML Type

Type is used to initialize XY-series coordinate data. 更多...

Import Statement: import QtCharts 2.1

属性

详细描述

XYPoint is a convenience element for initializing XY-series with static coordinate data. To manipulate an XY-series dynamically, use it's data manipulation functions instead.

参照 LineSeries, AreaSeries, ScatterSeries, and SplineSeries.

属性文件

The x-coordinate of the point.

The y-coordinate of the point.
