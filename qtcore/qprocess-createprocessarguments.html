(QProcess::CreateProcessArguments)

  • List of all members, including inherited members

详细描述

注:This struct is only available on the Windows platform.

This struct is a representation of all parameters of the Windows API function CreateProcess. It is used as parameter for CreateProcessArgumentModifier functions.

参照 QProcess::CreateProcessArgumentModifier.
