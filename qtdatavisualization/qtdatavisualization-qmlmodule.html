Qt Data Visualization QML Types

The Qt Data Visualization functionality can be accessed via these QML types.

QML Types

Abstract3DSeries

Base type for all QtDataVisualization series

AbstractAxis3D

Base type for axes of a graph

AbstractDataProxy

Base type for all QtDataVisualization data proxies

AbstractGraph3D

Base type for 3D visualizations

AbstractInputHandler3D

Base type for all QtDataVisualization input handlers

Bar3DSeries

Base series type for Bars3D

BarDataProxy

Base proxy type for Bars3D

Bars3D

3D bar graph

Camera3D

Representation of a camera in 3D space

CategoryAxis3D

Type is used for manipulating an axis of a graph

ColorGradient

Defines a color gradient

ColorGradientStop

Defines the color at a position in ColorGradient

Custom3DItem

Type is for creating custom items to be added to a graph

Custom3DLabel

Type is for creating custom labels to be added to a graph

Custom3DVolume

Type is for creating volume rendered objects to be added to a graph

HeightMapSurfaceDataProxy

Base proxy type for Surface3D

InputHandler3D

Basic wheel mouse based input handler

ItemModelBarDataProxy

Proxy class for presenting data in item models with Bars3D

ItemModelScatterDataProxy

Proxy class for presenting data in item models with Scatter3D

ItemModelSurfaceDataProxy

Proxy class for presenting data in item models with Surface3D

Light3D

Representation of a light source in 3D space

LogValueAxis3DFormatter

Implements logarithmic value axis formatter

Scatter3D

3D scatter graph

Scatter3DSeries

Base series type for Scatter3D

ScatterDataProxy

Base proxy class for Scatter3D

Scene3D

Type provides description of the 3D scene being visualized

Surface3D

Describes the usage of the 3D surface graph

Surface3DSeries

Base series type for Surfaces3D

SurfaceDataProxy

Base proxy class for Surface3D

Theme3D

A visual style for graphs

ThemeColor

Defines the color in a Theme3D

TouchInputHandler3D

Basic touch display based input handler

ValueAxis3D

Type is used for manipulating an axis of a graph

ValueAxis3DFormatter

Base type for value axis formatters
