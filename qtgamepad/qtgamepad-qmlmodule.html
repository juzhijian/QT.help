Qt Gamepad QML Types

The Qt Gamepad module provides QML types that can be used to access the state of gamepad hardware that may be available on a system.

The QML types can be imported into your application using the following import statement:


  import QtGamepad 1.0

To link against the module, add this line to your qmake .pro file:


  QT += gamepad

For more information about using these types in your application, see the Qt Gamepad documentation.
