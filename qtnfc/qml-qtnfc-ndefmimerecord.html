NdefMimeRecord QML Type

Represents an NFC MIME record. 更多...

Import Statement: import QtNfc 5.2
Since: Qt 5.2
Inherits:

NdefRecord

  • List of all members, including inherited members

属性

  • uri : string

详细描述

The NdefMimeRecord type can contain data with an associated MIME type. The data is accessible from the uri in the uri property.

属性文件

uri : string

This property hold the URI from which the MIME data can be fetched from. Currently this property returns a data url.
