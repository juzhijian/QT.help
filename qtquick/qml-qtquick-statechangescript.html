StateChangeScript QML Type

Specifies how to run a script in a state 更多...

Import Statement: import QtQuick 2.7
  • List of all members, including inherited members

属性

  • name : string
  • script : script

详细描述

A StateChangeScript is run upon entering a state. You can optionally use ScriptAction to specify the point in the transition at which the StateChangeScript should be run.


  State {
      name: "state1"
      StateChangeScript {
          name: "myScript"
          script: doStateStuff();
      }
      // ...
  }
  // ...
  Transition {
      to: "state1"
      SequentialAnimation {
          NumberAnimation { /* ... */ }
          ScriptAction { scriptName: "myScript" }
          NumberAnimation { /* ... */ }
      }
  }

参照 ScriptAction.

属性文件

name : string

This property holds the name of the script. This name can be used by a ScriptAction to target a specific script.

参照 ScriptAction::scriptName.


script : script

This property holds the script to run when the state is current.
