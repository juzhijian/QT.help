Qt Labs Calendar

The Qt Labs Calendar module provides a set of types that can be used to build calendars in Qt Quick.

QML Types

Calendar

A calendar namespace

CalendarModel

A calendar model

DayOfWeekRow

A row of names for the days in a week

MonthGrid

A grid of days for a calendar month

WeekNumberColumn

A column of week numbers

注:Types in the Qt.labs module are not guaranteed to remain compatible in future versions.
