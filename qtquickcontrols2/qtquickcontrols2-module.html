Qt Quick Controls 2 C++ Classes

Provides classes for setting up the controls from C++. 更多...

QQuickStyle

QQQuickStyle class allows configuring the application style

详细描述

The C++ types can be included into your application using the following include statement:


  #include <QtQuickControls2>

注:If you are using a few classes from this module, we recommend including those specific classes only instead of the module.

To link against the corresponding C++ libraries, add the following to your qmake project file:


  QT += quickcontrols2
