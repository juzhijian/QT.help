KeyboardRow QML Type

Keyboard row for keyboard layouts. More...

Detailed Description

Specifies a row of keys in the keyboard layout.

Property Documentation

keyWeight : real

Sets the key weight for all children keys.

The default value is inherited from the parent element in the layout hierarchy.


smallTextVisible : bool

Sets the smallTextVisible for all children keys.

The default value is inherited from the parent element in the layout hierarchy.

This QML property was introduced in QtQuick.VirtualKeyboard 2.0.
