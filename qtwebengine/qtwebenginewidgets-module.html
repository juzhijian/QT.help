详细描述

The Qt WebEngineWidgets module provides a web browser engine as well as C++ classes to render and interact with web content.

要包含模块类的定义，使用以下指令:


  #include <QtWebEngineWidgets>

To link against the module, add the following to your qmake project file:


  QT += webenginewidgets
